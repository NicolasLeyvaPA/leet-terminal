# Polymarket Markets Viewer

A React application that displays markets from the Polymarket API.

## Getting Started

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

The app will be available at `http://localhost:5173`

### Build for Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## Features

- Fetches markets from Polymarket API
- Displays markets in a responsive grid layout
- Refresh button to reload markets
- Error handling and loading states
- Modern, clean UI design
